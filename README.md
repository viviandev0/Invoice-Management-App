#  Invoice Management App | Stage 2 Full-Stack

A professional, full-stack invoice management solution built for seamless business tracking. This application allows users to create, manage, and track invoices with a focus on high-quality UI/UX, data persistence, and accessibility.

##  Core Features

### 1. Full CRUD Capabilities
* **Create:** Add new invoices with multiple line items, client details, and payment terms.
* **Read:** Dashboard overview of all invoices with a detailed individual view.
* **Update:** Edit existing invoices or transition them from "Draft" to "Pending."
* **Delete:** Remove invoices with a secure confirmation modal to prevent accidental data loss.

### 2. Intelligent Workflow
* **Status Management:** Flow through **Draft**, **Pending**, and **Paid** statuses.
* **Filtering:** Instantly filter your dashboard by status to find exactly what you're looking for.
* **Drafting:** Save your work-in-progress as a draft and finish it later.

### 3. Advanced UI/UX
* **Theme Toggle:** Switch between high-contrast **Light and Dark modes** (preference saved locally).
* **Responsive Design:** Optimized for Mobile (320px), Tablet (768px), and Desktop (1024px).
* **Interactive States:** Hover and active states for all buttons, inputs, and list items.

---

##  Tech Stack

* **Frontend:** React.js (Context API for State Management)
* **Styling:** CSS Modules / Tailwind CSS (Responsive & Accessible)
* **Persistence:** [LocalStorage / IndexedDB / Node.js Backend]
* **Validation:** Custom Form Validation for emails, required fields, and numeric logic.
